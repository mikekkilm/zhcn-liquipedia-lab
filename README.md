# zhcn-liquipedia-lab

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本项目主要用于收集、整理和发布多个独立的 HTML 页面。这些页面经过本地化处理或内容整理后，以静态文件的形式存储于此仓库中，方便后续查阅和使用。

本仓库不针对任何单一网站或域名，而是作为一个通用的 HTML 页面归档与发布空间。

## 目录说明

```
├── pages/          # 存放归档的 HTML 页面
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── docs/           # 文档或说明文件
└── README.md       # 本文件
```

各目录的具体用途：

- **pages/**：所有归档的 HTML 页面均存放于此。每个页面可能对应一个独立的主题或内容。
- **assets/**：与页面相关的资源文件，按照页面或功能分类存放。
- **docs/**：包含项目的说明文档、索引或更新日志等。

## 页面归档说明

归档的 HTML 页面通常满足以下特点：

- 内容为中文或包含中文翻译/本地化处理
- 以独立页面形式存在，无需依赖外部动态服务
- 经过基本的内容整理与格式调整
- 可离线或在静态服务器环境下直接打开查看

每个页面文件均包含完整的 HTML 结构，通常附带必要的资源引用，以保证在本地或静态环境下正常显示。

## 维护说明

本项目采用常规的 Git 仓库维护方式：

- 新增页面请放入 `pages/` 目录，并保持命名清晰
- 资源文件请放入 `assets/` 目录，避免与页面文件混淆
- 更新或修改页面内容后，建议在 commit 信息中简要说明变更内容
- 不鼓励在页面中包含外部动态资源或需要联网才能正常加载的内容

## 声明

本仓库仅作为技术性归档用途，不对所归档页面的内容作任何额外主张。所有页面内容均来自公开可获取的信息，经过整理后以静态形式重新发布，仅供学习与参考。
